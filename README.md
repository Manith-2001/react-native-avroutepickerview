# react-native-avroutepicker

iOS AVRoutePickerView for React Native

## Installation

```sh
npm install react-native-avroutepicker
```

## Usage

```js
import Avroutepicker from "react-native-avroutepicker";

// ...

const result = await Avroutepicker.multiply(3, 7);
```
